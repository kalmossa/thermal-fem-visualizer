Introduction

Ce projet présente les résultats de simulations thermiques réalisées sur cinq matériaux à changement de phase (MCP) : RT2, RT2HC, RT2_EG, Eutectic_0C et Hydrate_0C. Chaque simulation modélise le comportement thermique d'une couche de MCP soumise à une température extérieure variable, en faisant varier l'épaisseur de la couche, le coefficient de convection et la condition initiale de température.

Au total, 180 fichiers CSV ont été classés selon quatre critères : le type de MCP, l'épaisseur, la convection et la condition initiale. Trois cas représentatifs sont présentés sur cette page. L'interface est conçue pour rester lisible si de nouvelles colonnes apparaissent dans de futurs fichiers CSV — seules les colonnes garanties sont affichées.

Graphiques comparatifs

Chaque graphique compare les 5 MCP pour une même combinaison de paramètres. Les 33 autres graphiques se trouvent dans le dossier graphs/.

A — Cas standard

ep=15 mm, hx=1, dT0=+2°C
Configuration de référence. Les 5 MCP convergent progressivement vers une température négative après solidification complète.

B — Cas extrême

ep=10 mm, hx=0,5, dT0=−2°C
Condition initiale la plus froide, épaisseur et convection minimales. Aucun changement de phase — le MCP reste entièrement solide.

C — Cas optimal

ep=30 mm, hx=2, dT0=+4°C
Épaisseur maximale, convection forte, température initiale la plus haute. Meilleure capacité de stockage thermique parmi toutes les combinaisons.

Conclusion

Les simulations montrent que le comportement thermique des MCP varie significativement selon la condition initiale de température. Dans le cas extrême (dT0=−2°C), aucun changement de phase ne se produit et l'énergie restituée est quasi nulle. Dans le cas optimal (dT0=+4°C), le MCP Hydrate_0C se distingue par une capacité de stockage nettement supérieure aux autres matériaux.
